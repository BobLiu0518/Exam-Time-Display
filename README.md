# 班级电脑用考试时间展示

## 使用方法

1. 在班级电脑上打开本项目的 [GitHub Pages](https://exam.bobliu.tech/) 页面

   当然，你也可以选择把 index.html 下载到本地打开

2. 双击页面中间的提示，输入考试科目和时间（如：语文 08:00~11:30）
   
3. 按下`F11`，进入浏览器的全屏模式
